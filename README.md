# Nightscout Meal Photos

A visual food log that pulls meal treatments from Nightscout and displays photos inline.

Photos are uploaded to [freeimage.host](https://freeimage.host) and linked in Nightscout treatment notes.

## Features
- Timeline view of meals with photos
- Carb counts and meal descriptions
- Click-to-zoom lightbox
- Date range filtering
- Daily stats (meals, total carbs, photos)

## Setup
Requires CORS enabled on Nightscout (`cors` in the `ENABLE` env var).
